About

Ultimate YouTube Downloader is a feature-rich desktop GUI application for downloading YouTube videos, playlists, and audio tracks. It supports up to 8K resolution, merges best video+audio, embeds metadata and thumbnails, and optionally downloads subtitles.

Features

  • Download videos in best quality up to 8K/4320p
  • Extract high-quality MP3 audio with metadata
  • Full playlist support
  • Download English + auto-generated subtitles
  • Embed thumbnails and artwork
  • Live progress with percent, speed, and ETA
  • Smart output naming: Uploader - Title.ext

Quick Setup

Requires Python 3.9+ and FFmpeg in PATH. Run these commands:

# Install Python dependencies
pip install yt-dlp customtkinter Pillow

# Navigate to Yt_Downloader & run
cd Yt_Downloader
python main.py

FFmpeg: Download from gyan.dev and add bin folder to your PATH.

Usage

  • Paste YouTube URLs (one per line, playlists supported)
  • Choose format: Video (MP4) or Audio (MP3)
  • Set resolution, subtitles, and thumbnail options
  • Pick output folder & click "Start Epic Download"
