# Number Counter Animation Tool

A web-based tool to create animated number counter videos using HTML, CSS & JavaScript. No installation required — just open the HTML file in any browser.

## Features

- **Custom Range** — Set any start and end number
- **Text Color** — Pick any color for the number (color picker + hex input)
- **Background Color** — Choose any background color
- **Font Size** — Adjustable from 20px to 200px
- **Font Style** — 7 different font families to choose from
- **Animation Speed** — Control how fast the counter runs
- **End Text** — Add custom suffix like `+`, `Lakh`, `Cr` etc.
- **Quick Presets** — One-click buttons for 1K, 10K, 1L, 10L, 1Cr
- **Download as Video** — Export animation as `.webm` video file

## How to Use

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge)
2. Set your desired **Start Number** and **End Number**
3. Pick **Text Color** and **Background Color**
4. Adjust **Font Size** and **Animation Speed** using sliders
5. Click **Start** to preview the animation
6. Click **Download as Video** to save as `.webm` file
