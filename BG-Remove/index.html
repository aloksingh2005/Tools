About

BG-Remove is a modern, user-friendly desktop application that uses AI (U²-Net neural network) to remove backgrounds from images. Built with Python, CustomTkinter for a sleek dark UI, and Pillow for image handling.

Features

  • Clean dark UI using CustomTkinter
  • Live before/after preview side-by-side
  • One-click background removal with progress feedback
  • Local AI (offline) or remove.bg API mode
  • API key persistence across sessions
  • Export as transparent PNG
  • 100% private — all processing stays on your machine

Quick Setup

Requires Python 3.10–3.12. Run these commands in PowerShell:

# Navigate to BG-Remove folder
cd BG-Remove

# Create virtual environment & install
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

# Launch the app
python app.py

Usage

  • Click "Select Image" → Choose PNG, JPG, BMP, or GIF
  • Choose Local (offline AI) or API (remove.bg) mode
  • Click "Remove Background" → Wait a few seconds
  • Click "Save Image" → Export as transparent PNG
