# 🔄 Image Format Converter

A fast, lightweight web-based image format converter that runs entirely in your browser. Convert images between popular formats with professional quality - all client-side with zero server uploads.

## ✨ Features

### 🔄 Format Conversion
- **Supported Formats**: PNG, JPG/JPEG, WebP, AVIF, BMP, and HEIC/HEIF
- **Smart Conversion**: Optimized algorithms for each format
- **Quality Control**: Adjustable quality from 1-100% (auto-disabled for lossless formats like PNG/BMP)
- **Batch Processing**: Convert multiple images simultaneously
- **HEIC Support**: Automatically converts Apple's HEIC/HEIF images to JPEG

### 🎯 Core Features
- **Drag & Drop**: Intuitive file upload
- **Live Preview**: See converted results before downloading
- **Batch Download**: ZIP all converted images at once
- **Individual Downloads**: Download any image separately
- **Progress Tracking**: Real-time conversion progress
- **File Management**: Add/remove files easily
- **Size Statistics**: Track file size changes after conversion
- **Dark/Light Theme**: Eye-friendly themes for any preference

## 🚀 Getting Started

1. Open `index.html` in any modern web browser
2. Drag and drop images or click to upload
3. Select output format and quality
4. Click "Convert Images"
5. Download individual files or all as ZIP

### No Installation Required
- **100% Client-Side**: All processing happens in your browser
- **No Server**: Your images never leave your device
- **Privacy First**: Complete data security
- **Zero Dependencies**: Works offline after first load

## 🛠️ Technical Details

- **HTML5**: Semantic, accessible markup
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **Vanilla JavaScript**: No framework dependencies
- **Libraries**:
  - `jszip` - for batch downloading
  - `FileSaver.js` - for client-side downloads
  - `heic2any` - for HEIC format support

## 📊 Performance
- **Lightweight**: ~500 lines of clean, optimized code
- **Fast**: Instant conversion for most images
- **Responsive**: Works on desktop, tablet, and mobile
- **Smooth**: No lag or heavy scrolling issues

## 🔐 Privacy & Security
All processing is local. No data is collected or uploaded to any server. Completely GDPR compliant by design.
